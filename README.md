## Requirements:
- ```pip install flask```
- ```pip install jwt```
- ```pip install PyJWT```
---
## How to Run:
1. ```cd code```
2. ```.\venv\Scripts\activate```
3. ```flask run```

When JWT Token is generated upon logging in (in Postman):
Go to the Headers tab
Add a key called: x-access-token
Paste your token as the value
